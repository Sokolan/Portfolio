Battleships

The "Battleship" project is a completed JavaScript venture that showcases skills in object-oriented programming and Test-Driven Development (TDD). It features a recreation of the classic Battleship game, integrating classes for ships, gameboards, and players. Key components include a Ship class with hit and sunk functionalities, a Gameboard class for handling attacks, and a Player class for turn-based gameplay. The project is completed with an interactive user interface, highlighting the developer's proficiency in JavaScript, TDD, and interactive design.

The Weather App project is a JavaScript development exercise focusing on API integration and asynchronous programming. It involves creating a weather forecast application using a weather API, allowing users to search for locations and toggle between Fahrenheit and Celsius. The app's interface changes based on the weather data, potentially integrating weather-related images or gifs. ToDo List

The Todo List project is a JavaScript-based exercise focused on creating a dynamic, feature-rich to-do list application. It includes the implementation of object-oriented programming for creating todo items, with properties like title, description, due date, and priority. The app organizes todos into projects, with default and user-created options. It features a clean user interface for viewing, editing, and deleting todos, and employs local storage for data persistence.
